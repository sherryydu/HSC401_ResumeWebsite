## Project description/purpose

I created a static website page to display my resume. This project was completed in fulfillment of the requirements of HSC401H5: Health and Science Communication Design in Fall 2021.

This root directory ('HSC401_ResumeWebsite') contains folders and an HTML file related to this project for the website's functionality.

## How to use:

This project is intended to be run locally on your desktop.

1. Download the root directory ('HSC401_ResumeWebsite')
2. Open the 'index.html' page in your chosen web browser (e.g., Google Chrome, Safari, etc.).
3. Enjoy!
